Precious

A Gollum-themed programming language that compiles to C.

github.com/korrykatti/precious

Install

cmake -B build && cmake --build build
./build/precious your_file.precious
./out

Outputs a Linux ELF binary named out via gcc. The compiler generates C source code and lets gcc handle register allocation, optimization, and code generation.

Variables

Declare with we_haves. Assign with =.

we_haves x = 42;
x = 10;

Type Annotations

Optional : type after the variable name. Inferred if omitted.

PreciousC typeDescription
numberlong64-bit integers
wordconst char*Strings
questionlongBooleans
decimaldoubleFloating-point
lettercharSingle characters
we_haves x: number = 5;
we_haves name: word = "gollum";
we_haves y = 10;              // inferred as number
we_haves msg = "hello";       // inferred as word

Operators

Arithmetic

we_haves result = 2 + 3 * 4 - 1;  // 13
gives(result);

Comparison

== != < > <= >= вҖ” return 1 or 0.

Boolean

and, or, ! вҖ” precedence: ! > and > or.

if (x > 0 and x < 10) { gives(1); }
if (!0) { gives(2); }
if (x == 1 or x == 5) { gives(3); }

If / Elif / Else

we_haves x = 5;
if (x == 5) {
    gives(10);
} elif (x > 3) {
    gives(20);
} else {
    gives(30);
}

While Loop

we_haves i = 0;
while (i < 5) {
    i = i + 1;
}
gives(i);

Scopes

we_haves x = 5;
{
    we_haves y = 10;
    gives(x + y);
}

Print

say(expr) prints to stdout. Works with integers, string literals, and string variables.

say(42);          // prints 42
say("hello");     // prints hello

we_haves msg: word = "gollum";
say(msg);         // prints gollum

Functions

Define with fn. Return with gives. Parameters support type annotations.

fn add(a, b) {
    say(a + b);
}
add(2, 3);          // prints 5

fn greet(name: word) {
    say(name);
}
greet("precious");  // prints precious

fn multiply(a, b) {
    gives(a * b);
}
we_haves result = multiply(6, 7);
say(result);        // prints 42

Return type is auto-detected. Functions can call each other regardless of declaration order.
